# code split
webpack是一个前端代码打包工具，前端代码需要在客户端运行，所以多数情况需要考虑性能优化。性能优化有很多途径，比如使用CDN加载静态资源，使用缓存等等。这里只讨论和性能优化相关的两种情况：1. 前端应当尽量减少请求次数 2. 单一资源的响应体不应太大。webpack的code split就是解决上面两个问题。
